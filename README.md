Morse code is a method of sending text using short and long signals, traditionally called dots (·) and dashes (–). It was developed in the 1830s and 1840s by Samuel Morse and Alfred Vail for use with the telegraph.

Each letter, number, and some punctuation marks have their own unique pattern. For example:

A = · –
B = – · · ·
C = – · – ·
S = · · ·
O = – – –
SOS = · · · – – – · · ·

Morse code can be sent in many ways:

Sound (short and long beeps)
Light (short and long flashes)
Radio signals
Taps or knocks

History of Morse Code

Samuel Morse developed the electric telegraph in the 1830s. Together with Alfred Vail, he created the first version of Morse code.

Timeline
1837 – Samuel Morse developed the electric telegraph.
1838 – The first Morse code system was demonstrated.

1844 – The first official telegraph message was sent:

"What hath God wrought?"

1851 – International Morse Code was introduced in Europe.
1865 – International Morse Code became widely adopted.
1999 – Morse code was removed as a mandatory requirement for many radio operator licenses in several countries.
